Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Title is is expremely important for SEO.
  • Meta contains the information about the page that is used by the browser.
  • Elements refers to the entire element from the opening tag to the closing tag.
  • HTML elements are case sensitive
  • A declaration is not case sensitive
  • HTML attributes define additional instruction about the element.They are always defined in the beginning of the element never the closing.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding idicates how much space we want around the content inside an element.
  • A sector defines the element or attributes to which the rules or declarations apply.
  • Eternal CSS style sheet keeps all their CSS rules in a seperate file. It is the most common way to style a webpage
  • Declarations contain 2 important components: The CSS property we want to apply and the value of the property.
  • * - applies a rule to all elements visible on a page.
  • Assigning colors in CSS can be done in many ways, including by semantic references and hexadecimal code, or hex#.
